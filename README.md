# Big-Beautiful-Board
This repository is for the ITEK development Big Beuatiful Board for Raspberry Pi Pico. Note Big Beautiful Boards  coding language is primarily Micropython (and maybe a bit of Python. But if Python is used, it will be clarified).

### Simulations
To simulate Raspberry Pi Pico before applying it to real world scenarios on the Big Beuatiful Board. In ITEK will use "Wokwi" for simulations. 
- https://wokwi.com/
